MODULE MKAPI.CORE.DOCSTRING

This module provides functions that parse docstring.

Functions
  • section_heading(line) (str, str) — Returns a tuple of (section name, style name).
  • split_section(doc) (str, str, str) — Yields a tuple of (section name, contents, style).
  • split_parameter(doc) (list of str) — Yields a list of parameter string.
  • parse_parameter(lines,style) (str, str, str) — Returns a tuple of (name, markdown, type).
  • parse_parameters(doc,style) (list of (str, str, str)) — Returns a list of (name, markdown, type).
  • parse_returns(doc,style) (str, str) — Returns a tuple of (markdown, type).
  • get_section(name,doc,style) (Section) — Returns a Section instance.
  • parse_bases(doc,obj) — Parses base classes to create a Base(s) line.
  • parse_property(doc,obj) — Parses property's docstring to inspect type.
  • parse_attribute(doc,obj) — Parses attributes' docstring to inspect type and description from source.
  • get_docstring(obj) (Docstring) — Returns a Docstring instance.

FUNCTION mkapi.core.docstring.section_heading(line) → (str, str)

Returns a tuple of (section name, style name).

Parameters
  • line (str) — Docstring line.
Examples 
>>> section_heading("Args:")
('Args', 'google')
>>> section_heading("Raises")
('Raises', 'numpy')
>>> section_heading("other")
('', '')

GENERATOR mkapi.core.docstring.split_section(doc) → (str, str, str)

Yields a tuple of (section name, contents, style).

Parameters
  • doc (str) — Docstring
Examples 
>>> doc = "abc\n\nArgs:\n    x: X\n"
>>> it = split_section(doc)
>>> next(it)
('', 'abc', '')
>>> next(it)
('Parameters', 'x: X', 'google')
