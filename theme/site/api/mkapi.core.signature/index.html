FUNCTION mkapi.core.signature.to_string(annotation, kind='returns') → str

Returns string expression of annotation.

If possible, type string includes link.

Parameters
  • annotation — Annotation
  • kind (str, optional) — 'returns' or 'yields'
Examples 
>>> from typing import Iterator, List
>>> to_string(Iterator[str])
'iterator of str'
>>> to_string(Iterator[str], 'yields')
'str'
>>> from mkapi.core.node import Node
>>> to_string(List[Node])
'list of [Node](!mkapi.core.node.Node)'

FUNCTION mkapi.core.signature.a_of_b(annotation) → str

Returns A of B style string.

Parameters
  • annotation — Annotation
Examples 
>>> from typing import List, Iterable, Iterator
>>> a = List[str]
>>> a_of_b(a)
'list of str'
>>> a = Iterable[int]
>>> a_of_b(a)
'iterable of int'
>>> a = Iterator[float]
>>> a_of_b(a)
'iterator of float'

FUNCTION mkapi.core.signature.union(annotation) → str

Returns a string for union annotation.

Parameters
  • annotation — Annotation
Examples 
>>> from typing import List, Optional, Tuple, Union
>>> a = Optional[List[str]]
>>> union(a)
'list of str, optional'
>>> a = Union[str, int]
>>> union(a)
'str or int'
>>> a = Union[str, int, float]
>>> union(a)
'str, int, or float'
>>> a = Union[List[str], Tuple[int, int]]
>>> union(a)
'Union(list of str, (int, int))'
